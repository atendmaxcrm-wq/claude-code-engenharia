Investigue o seguinte problema reportado: $ARGUMENTS

Siga estes passos:

1. **Entender o problema**: Leia o arquivo de troubleshooting (`/memoria/sistema/troubleshooting.md`) para ver se ja e um problema conhecido
2. **Localizar codigo**: Identifique os arquivos envolvidos consultando os arquivos de memoria relevantes
3. **Analisar causa raiz**: Leia os arquivos de codigo e trace o fluxo de dados
4. **Verificar logs**: Cheque logs relevantes para erros
5. **Propor solucao**: Apresente a causa raiz e a correcao sugerida

NAO aplique correcoes automaticamente. Apresente o diagnostico e aguarde aprovacao.
